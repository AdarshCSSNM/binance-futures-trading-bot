Clone the Repository

Bash
git clone https://github.com/AdarshCSSNM/binance-futures-trading-bot.git
cd binance-futures-trading-bot
Install Dependencies
Ensure you have Python 3.8+ installed.

Bash
pip install -r requirements.txt
Configure Credentials
Duplicate the example environment file and add your Testnet API keys:

Bash
cp .env.example .env
Open .env and insert your BINANCE_API_KEY and BINANCE_API_SECRET.
